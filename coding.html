Humble beginnings, my first time learnng to code in HTML5. Of course, this is about as basic as a website can come just some arbitrary text on a white background.

Moving on to the second task. This is the first time I had experimented with CSS and FlexBox.

Continuing on with this task, I learnt the basics of alignment and justification within FlexBox. In addition to this I also learnt to create Lists, how to change background colour, how to style certain sections of text/body's/images, etc...

Also during this task I learnt to create hyperlinks within HTML and how to style said hyperlinks with CSS. I found this task was very useful and an easy step up from the previous. It taught me to basics of creating a half-decent looking webpage.

Moving on, during this task I learnt to use FlexBox to further align images and paragraphs. I used margins to aid in the alignment of the images and paragraphs.

Finally, for this task I created a basic footer that provided a short conclusion of the task.

Advancing quickly, during this task I further learnt more about HTML & CSS. This was thefirst time I made use of buttons with HTML and used them to act as anchor links within the page - one of the buttons acting as a hyperlink.

In addition to learning how to create buttons, I also learnt to add embedded videos inside of the webpage as seen above.

Similarly I also learnt to embed an MP3 file.

In the above images and much through my future projects I made use of Placeholder images - these were useful to figure out how big I want my images in Px wihout any guesswork.

This was some more practice for linking different webpages within the same project using hyperlinks within a button.

This was my first time ever taking code from an external source. I learnt to take code from another website and add it to my own and how to ensure it worked by the 'Format Document' inside of VSc.

I added the code to create a feature that allowed for multiple images to be displayed each time you press a different number/move the slider left & right.
