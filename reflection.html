Reflection

Whilst creating this project I have encountered many issues, successes and frustrations. This being my first large project I have ever coded I believe it to be only natural that I encountered issues. I have found that by persevering, researching and seeking aid I think I have been able to create a final product that not only works, but one that I have found great enjoyment creating, learning about and applying various problem solving skills to fix. Some examples of this: Responsitivity, at first I found I had a lot of trouble getting this site to act responsivly with screens of different sizes and as a result it haltered my progress until I could understand what I was doing wrong (That being an excessive use of Padding rather than using tools available to me to move my Contents Areas). In addition to issues with Responsitivity, I found understanding all the different functions within HTML, CSS and Flexbox quite the daunting task and at first it took me a decent while to get the ball rolling.

Despite the issues I encountered I think I learned a lot about how to create a basic website, how to problem solve in topics that are completley alien to me, and how to experimentaly see what each function does to see what works and what doesn't. If I was to change anything about this website I think 100% it would have been to take a step back and fully understand Flexbox before going Gung Ho and jumping straight into creating this site - it would have saved me hours in the long run and many frustrations when it comes fixing issues with Responsitivity. If I had to summarise my journey in just a few words I'd describe it as frustrating, but very very rewarding once I managed to get everything working how I wanted.
